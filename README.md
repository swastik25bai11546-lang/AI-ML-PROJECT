The Movie Recommendation System is a machine learning-based application designed to suggest movies to users based on their preferences. It uses a content based filtering approach where recommendations are generated by analyzing similarities between movies. Movie Recommendation System

Hey! This is a simple Movie Recommendation System I built using Python and basic machine learning concepts. The goal of this project is to suggest movies based on what a user likes.It is a lightweight, highly interactive terminal-based movie recommendation engine built with Python. This project uses Natural Language Processing (NLP) to understand movie features and calculate similarities, providing highly accurate, personalized movie suggestions.

✨ Features Smart Content-Based Filtering: Uses TF-IDF Vectorization and Cosine Similarity to recommend movies based on a combination of Genre, Director, Cast, Language, and Year. Typo Forgiveness: Automatically detects and corrects misspelled movie names using the difflib library. Category Browsing: Allows users to filter and search the database by specific categories (e.g., finding all movies by a specific director). Interactive Terminal UI: Features a beautiful, easy-to-use, arrow-key-navigable menu powered by questionary.

🛠️ Prerequisites Make sure you have Python 3.x installed on your system. You will also need to install the following required libraries:

Python libraries used : 1- NumPy: Used for handling arrays and calculations. 2-Pandas: Used for managing the movie dataset, such as loading data from the CSV file and cleaning it before processing. 3-Scikit-learn (sklearn): Used for calculating the similarity between movies. Specifically, the code imports CountVectorizer to convert genre text into numerical vectors and cosine_similarity to create the similarity matrix.

🚀 How to Run Ensure the dataset file (movies.csv) is in the same directory as your Python script.

Run the script from your terminal:

Bash python main.py (Note: Replace main.py with the actual name of your python file if it is different).

Follow the on-screen interactive prompts to find recommendations or browse the catalog!

📂 Dataset Requirements (movies.csv) For the program to work correctly, your CSV file should ideally contain the following columns:

Movie Name,Genre,Director,Cast,Language, Year

(Note: The program handles missing data automatically, but having these columns yields the best AI recommendations).
